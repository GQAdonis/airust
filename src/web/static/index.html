Agent Types
The brain of AIRust — choose the right agent for your use case
MatchAgent
The simplest and fastest agent. Compares questions directly against training data using exact or fuzzy matching with Levenshtein distance.
let agent = MatchAgent::new_fuzzy();
agent.train(&knowledge);
let answer = agent.predict("hello");
Best for: FAQ bots, command recognition, structured Q&A
TfidfAgent
Semantic search using TF-IDF and BM25 ranking. Understands meaning, not just words. Handles synonyms and related concepts.
let agent = TfidfAgent::new();
agent.train(&knowledge);
let answer = agent.predict("greeting");
Best for: Natural language Q&A, documentation search
ContextAgent
Wraps any agent and adds conversation memory. Maintains context across messages for multi-turn dialogue.
let agent = ContextAgent::new(inner);
agent.add_context("previous question");
let answer = agent.predict("and this?");
Best for: Chatbots, support agents, interactive assistants

Get Started
Up and running in seconds
# Add to your Cargo.toml
[dependencies]
airust = "0.1.7"

# Or run directly
cargo install airust
cargo run --release
# Build and run with Docker
docker build -t airust .
docker run -p 7070:7070 airust
# Clone and build from source
git clone https://github.com/LEVOGNE/airust.git
cd airust
cargo build --release
cargo run --release
